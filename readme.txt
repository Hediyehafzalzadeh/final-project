** راهنمای راه اندازی : 

 این پروژه شامل دو بخش front-end و back-end هست که در دو فایل از هم جدا هستند و توسط API  به هم متصل شده اند.
 
------------------------------------------------------------------
**برای راه اندازی بخش front-end :

1  -  npm install 
2  -  npm start 
------------------------------------------------------------------
**برای راه اندازی بخش back-end :

1  -   composer install
2 -  php artisan key:generate
3 -  php artisan serve
 
-------------------------------------------------------------------
** برای وارد کردن محصولات نمایشی فایل دیتا بیس را وارد کنید. 

** IMPORT کردن دیتا بیس :

1 -  http://localhost/phpmyadmin/

2 - USERNAME = root   PASSWORD IS EMPTY
